# VoIP Spike Report — HappyTails

**Статус:** spike завершён. Production-реализация — Stage 1.

---

## Что тестировалось

Текущая реализация: `/video.html?id=<consultationId>&role=vet|client` — страница на базе
браузерного WebRTC (RTCPeerConnection) с сигнальным каналом через WebSocket.

### Устройства
- Client: Chrome 124 / macOS + Android Chrome 124
- Vet: Safari 17.4 / macOS

### Сеть
- Оба на одном WiFi (LAN) — тест 1
- Client через мобильный 4G, Vet на WiFi — тест 2
- Оба через мобильный 4G (UCell, Tashkent) — тест 3

---

## Результаты

| Сценарий | WebRTC P2P | Качество | Проблемы |
|---|---|---|---|
| LAN ↔ LAN | ✅ Работает | Хорошее | — |
| 4G ↔ WiFi | ⚠️ Непостоянно | Среднее | Иногда не устанавливается ICE |
| 4G ↔ 4G | ❌ Не работает | — | Symmetric NAT, нет TURN |

### NAT traversal без TURN: ❌
Операторы Узбекистана (UCell, Ucell, Beeline UZ, Humans) используют Carrier-Grade NAT (CGNAT).
Прямое P2P соединение работает только при LAN/WiFi. Для мобильного трафика **TURN-сервер обязателен**.

### Совместимость браузеров
- Chrome/Firefox: ✅
- Safari 17+: ✅ (getUserMedia требует HTTPS)
- Telegram WebApp браузер: ⚠️ — getUserMedia требует явного разрешения, в iOS ограничен

## Оценка: Daily.co vs Agora vs self-hosted TURN

### Daily.co
- **Плюсы:** готовый SDK, работает из коробки, TURN включён, HIPAA-ready
- **Минусы:** серверы в EU/US — latency ~180ms из Ташкента; $0.004/мин при масштабе
- **Соответствие УЗ:** данные проходят через иностранные серверы — потенциально проблема с локализацией данных по 2021 UZ data law

### Agora
- **Плюсы:** PoP в Казахстане (близко к UZ), latency ~60-80ms; SDK зрелый
- **Минусы:** цена при объёме ($0.00099/мин/пользователь); сложная тарификация; китайская компания
- **Соответствие УЗ:** требует проверки в юридическом отделе

### Self-hosted Coturn (TURN)
- **Плюсы:** полный контроль, данные остаются в УЗ, минимальный relay cost
- **Минусы:** нужен VPS в Ташкенте (~$20-40/мес), ops-нагрузка, нет мониторинга quality
- **Соответствие УЗ:** ✅ — данные на местной инфраструктуре

---

## Рекомендация

**Stage 1:** self-hosted Coturn на Ташкентском VPS (UCloud.uz или Beget.uz) + существующий
WebRTC сигнальный канал через наш WebSocket. Это закрывает CGNAT-проблему, данные остаются в УЗ,
стоимость ~$30/мес при 500 консультациях.

**Stage 2 (при росте > 1000 консультаций/мес):** оценить Agora с учётом latency-тестов из Ташкента.
